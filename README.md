# toolsrcipt
脚本管理
move_jpg.py
        功能:
        根据txt查找相同名字的jpg
        用法：
        1.输入总的路径（里面包含.txt文件以及.jpg文件）
          脚本会自动检查里面的txt和jpg检查完后根据txt来查找相同名字的jpg
remove_txt.py
        功能:
        删除路径下面的所有txt文件
        用法:
        输入文件总路径
txt_move.py
        功能:
        根据输入的内容查找txt文件里面有没有对应的内容如果有则新建一个文件夹
        把相同具有相同属性txt移动到同一个文件夹里面
        用法:
        1.输入文件总路径
        2.输入属性名称
